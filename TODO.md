# Run Brightbot Project

## Steps to Run the Project

- [x] Install backend dependencies: pip install -r requirements.txt
- [ ] Install frontend dependencies: cd frontend && npm install
- [x] Start backend server: cd backend && python main_new.py
- [x] Start frontend server: cd frontend && npm start
- [ ] Verify servers are running on ports 8001 (backend) and 3000 (frontend)
- [ ] Open browser to http://localhost:3000 to access the frontend
